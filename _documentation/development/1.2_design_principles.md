# 247 Terminal - Design Principles & Visual Direction

## Design Philosophy

**Core Concept:** Premium fintech trading platform that feels fast, trustworthy, and cutting-edge.

**Keywords:** Sleek, Modern, Premium, Fast, Trustworthy, Professional

**Inspiration:** Stripe, Robinhood, Linear, Vercel, Raycast

---

## What We're Changing From the Current Site

| Current Site | New Direction |
|--------------|---------------|
| Generic Webflow template feel | Custom, premium fintech aesthetic |
| Flat sections stacked vertically | Bento grid layouts with visual hierarchy |
| Basic gold glow effect | Sophisticated gradients + glassmorphism |
| Standard card layouts | Interactive, elevated card system |
| Static imagery | Subtle animations + micro-interactions |
| Dense text blocks | Generous whitespace, scannable content |
| Generic dark background | Layered depth with surface elevation |

## Color System

### Background Layers (Not Pure Black)

Pure black (#000000) is harsh on screens. We use layered dark grays with subtle blue undertones for depth and elegance.

```css
/* Background Hierarchy */
--bg-base: #09090b;        /* Deepest background (zinc-950) */
--bg-surface: #18181b;     /* Cards, sections (zinc-900) */
--bg-elevated: #27272a;    /* Hover states, modals (zinc-800) */
--bg-overlay: #3f3f46;     /* Tooltips, dropdowns (zinc-700) */
```

### Accent Colors

```css
/* Primary Accent - Gold/Amber */
--accent-primary: #f59e0b;      /* Amber-500 - main accent */
--accent-primary-light: #fbbf24; /* Amber-400 - hover states */
--accent-primary-dark: #d97706;  /* Amber-600 - active states */

/* Accent Glow */
--accent-glow: rgba(245, 158, 11, 0.15);  /* Subtle glow */
--accent-glow-strong: rgba(245, 158, 11, 0.3); /* Prominent glow */

/* Secondary Accent - For variety */
--accent-secondary: #3b82f6;  /* Blue-500 - links, info */
```

### Text Colors (Not Pure White)

Pure white (#FFFFFF) on dark backgrounds causes eye strain. We use softer whites.

```css
/* Text Hierarchy */
--text-primary: #fafafa;      /* High emphasis (zinc-50) */
--text-secondary: #a1a1aa;    /* Medium emphasis (zinc-400) */
--text-tertiary: #71717a;     /* Low emphasis (zinc-500) */
--text-disabled: #52525b;     /* Disabled state (zinc-600) */
```

### Semantic Colors

```css
/* Status Colors */
--success: #22c55e;   /* Green-500 */
--warning: #f59e0b;   /* Amber-500 */
--error: #ef4444;     /* Red-500 */
--info: #3b82f6;      /* Blue-500 */
```

### Gradient Definitions

```css
/* Hero gradient - dramatic top-down */
--gradient-hero: linear-gradient(
  180deg,
  rgba(245, 158, 11, 0.08) 0%,
  transparent 50%
);

/* Card gradient - subtle shine */
--gradient-card: linear-gradient(
  135deg,
  rgba(255, 255, 255, 0.03) 0%,
  transparent 50%
);

/* Accent gradient - for buttons/highlights */
--gradient-accent: linear-gradient(
  135deg,
  #f59e0b 0%,
  #d97706 100%
);

/* Mesh gradient - for backgrounds */
--gradient-mesh: radial-gradient(
  at 50% 0%,
  rgba(245, 158, 11, 0.1) 0%,
  transparent 50%
);
```

## Typography

### Font Stack

```css
/* Primary: Inter - modern, variable, excellent readability */
--font-sans: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* Mono: For code/data displays */
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```

### Type Scale

```css
/* Fluid typography with clamp() */
--text-xs: clamp(0.75rem, 0.7rem + 0.25vw, 0.875rem);     /* 12-14px */
--text-sm: clamp(0.875rem, 0.8rem + 0.35vw, 1rem);        /* 14-16px */
--text-base: clamp(1rem, 0.9rem + 0.5vw, 1.125rem);       /* 16-18px */
--text-lg: clamp(1.125rem, 1rem + 0.6vw, 1.25rem);        /* 18-20px */
--text-xl: clamp(1.25rem, 1.1rem + 0.75vw, 1.5rem);       /* 20-24px */
--text-2xl: clamp(1.5rem, 1.2rem + 1.5vw, 2rem);          /* 24-32px */
--text-3xl: clamp(1.875rem, 1.5rem + 1.9vw, 2.5rem);      /* 30-40px */
--text-4xl: clamp(2.25rem, 1.8rem + 2.25vw, 3rem);        /* 36-48px */
--text-5xl: clamp(3rem, 2.2rem + 4vw, 4.5rem);            /* 48-72px */
```

### Font Weights

```css
--font-normal: 400;    /* Body text */
--font-medium: 500;    /* Emphasized body, labels */
--font-semibold: 600;  /* Subheadings, buttons */
--font-bold: 700;      /* Headlines */
```

### Typography Rules

1. **Headlines**: Bold (700), tight letter-spacing (-0.02em)
2. **Body**: Normal (400) or Medium (500), relaxed line-height (1.6-1.75)
3. **Labels/UI**: Medium (500), slight letter-spacing (0.01em)
4. **All caps**: Only for small labels, always with letter-spacing (0.05em)

## Visual Effects

### Glassmorphism (Subtle)

Used sparingly for cards, modals, and navigation.

```css
.glass {
  background: rgba(24, 24, 27, 0.7);
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.05);
}

.glass-accent {
  background: rgba(245, 158, 11, 0.05);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(245, 158, 11, 0.1);
}
```

### Card Elevation System

```css
/* Level 0 - Flat on background */
.card-flat {
  background: var(--bg-surface);
  border: 1px solid rgba(255, 255, 255, 0.05);
}

/* Level 1 - Slight elevation */
.card-elevated {
  background: var(--bg-surface);
  border: 1px solid rgba(255, 255, 255, 0.08);
  box-shadow:
    0 1px 2px rgba(0, 0, 0, 0.3),
    0 4px 8px rgba(0, 0, 0, 0.2);
}

/* Level 2 - Prominent (featured cards) */
.card-prominent {
  background: linear-gradient(
    135deg,
    rgba(255, 255, 255, 0.05) 0%,
    rgba(255, 255, 255, 0.02) 100%
  );
  border: 1px solid rgba(255, 255, 255, 0.1);
  box-shadow:
    0 4px 6px rgba(0, 0, 0, 0.3),
    0 10px 20px rgba(0, 0, 0, 0.25);
}

/* Accent Card - Gold highlight */
.card-accent {
  background: var(--bg-surface);
  border: 1px solid rgba(245, 158, 11, 0.3);
  box-shadow:
    0 0 0 1px rgba(245, 158, 11, 0.1),
    0 4px 20px rgba(245, 158, 11, 0.15);
}
```

## Micro-Interactions & Animation

### Timing Functions

```css
--ease-out: cubic-bezier(0.16, 1, 0.3, 1);      /* Smooth deceleration */
--ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);  /* Balanced */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1); /* Bouncy */
```

### Duration Scale

```css
--duration-fast: 150ms;    /* Hover states */
--duration-normal: 250ms;  /* Most transitions */
--duration-slow: 400ms;    /* Page transitions */
--duration-slower: 600ms;  /* Complex animations */
```

### Hover Effects

```css
/* Card hover - lift effect */
.card-interactive {
  transition: transform var(--duration-normal) var(--ease-out),
              box-shadow var(--duration-normal) var(--ease-out);
}
.card-interactive:hover {
  transform: translateY(-4px);
  box-shadow:
    0 8px 16px rgba(0, 0, 0, 0.3),
    0 16px 32px rgba(0, 0, 0, 0.2);
}

/* Button hover - glow intensify */
.btn-primary:hover {
  box-shadow: 0 0 30px rgba(245, 158, 11, 0.4);
}

/* Link hover - underline animation */
.link-animated {
  position: relative;
}
.link-animated::after {
  content: '';
  position: absolute;
  bottom: -2px;
  left: 0;
  width: 0;
  height: 2px;
  background: var(--accent-primary);
  transition: width var(--duration-normal) var(--ease-out);
}
.link-animated:hover::after {
  width: 100%;
}
```

### Scroll Animations

Elements animate in as they enter the viewport:

```css
/* Fade up on scroll */
.animate-on-scroll {
  opacity: 0;
  transform: translateY(20px);
  transition:
    opacity var(--duration-slow) var(--ease-out),
    transform var(--duration-slow) var(--ease-out);
}
.animate-on-scroll.visible {
  opacity: 1;
  transform: translateY(0);
}

/* Stagger children */
.stagger-children > * {
  transition-delay: calc(var(--index, 0) * 100ms);
}
```

## Imagery Guidelines

### Screenshots

- Frame with subtle glass border
- Add soft shadow beneath
- Consider adding subtle reflection effect
- Crop to show the most impactful parts

### Icons

- Use consistent stroke width (1.5-2px)
- Gold accent color for feature icons
- White/gray for UI icons
- Consider phosphor-icons or lucide-icons

### Backgrounds

- Use gradient meshes, not solid colors
- Subtle noise texture (optional, very subtle)
- Abstract geometric shapes with low opacity

---

## Accessibility Requirements

### Contrast Ratios

| Element | Minimum Ratio |
|---------|---------------|
| Body text | 4.5:1 |
| Large text (18px+) | 3:1 |
| UI components | 3:1 |
| Focus indicators | 3:1 |

### Focus States

```css
/* Visible focus for keyboard navigation */
:focus-visible {
  outline: 2px solid var(--accent-primary);
  outline-offset: 2px;
}
```

### Reduced Motion

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
