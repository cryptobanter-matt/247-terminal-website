# 247 Terminal Website - Detailed Task Breakdown

## Phase 1: Project Setup

### 1.1 Initialize Astro Project

**Command:**
```bash
npm create astro@latest . -- --template minimal --typescript strict
```

**Configuration options:**
- Template: minimal (cleanest starting point)
- TypeScript: strict
- Install dependencies: yes

**Post-init tasks:**
- [ ] Verify `astro.config.mjs` is created
- [ ] Confirm `tsconfig.json` exists with strict mode

### 1.2 Install Dependencies

**Core dependencies:**
```bash
npm install @astrojs/preact preact
npm install -D tailwindcss @astrojs/tailwind
```

**Optional (for animations):**
```bash
npm install motion
```

### 1.3 Configure Astro for Preact

**Update `astro.config.mjs`:**
```javascript
import { defineConfig } from 'astro/config';
import preact from '@astrojs/preact';
import tailwind from '@astrojs/tailwind';

export default defineConfig({
  integrations: [
    preact({ compat: true }),
    tailwind()
  ],
  output: 'static'
});
```

### 1.4 Configure Tailwind CSS

**Create `tailwind.config.mjs`:**
```javascript
export default {
  content: ['./src/**/*.{astro,html,js,jsx,md,mdx,svelte,ts,tsx,vue}'],
  theme: {
    extend: {
      colors: {
        background: '#0a0a0a',
        surface: '#1a1a1a',
        accent: '#efb90a',
        'accent-glow': 'rgba(239, 185, 10, 0.3)',
      },
      fontFamily: {
        sans: ['Inter', 'Open Sans', 'system-ui', 'sans-serif'],
      },
    },
  },
  plugins: [],
};
```

### 1.5 Project Structure

Create the following directory structure:
```
src/
├── components/
│   ├── common/          # Shared components (Button, Card, etc.)
│   ├── layout/          # Header, Footer, Navigation
```

### 2.3 Create Mobile Menu (Interactive Island)

**File: `src/components/interactive/MobileMenu.tsx`**

```tsx
import { useState } from 'preact/hooks';

export default function MobileMenu() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div class="md:hidden">
      <button onClick={() => setIsOpen(!isOpen)}>
        {/* Hamburger icon */}
      </button>
      {isOpen && (
        <nav class="absolute top-full left-0 w-full bg-surface">
          {/* Mobile nav links */}
        </nav>
      )}
    </div>
  );
}
```

**Usage in Header.astro:**
```astro
---
import MobileMenu from '../interactive/MobileMenu.tsx';
---
<MobileMenu client:load />
```

### 8.1 Build Configuration

**Verify `astro.config.mjs`:**

```javascript
export default defineConfig({
  site: 'https://247terminal.com',
  output: 'static',
  // ... other config
```

### 8.2 Build & Test Locally

```bash
npm run build
npm run preview
```

**Verify:**
- [ ] All pages render correctly
- [ ] No console errors
- [ ] Images load properly
- [ ] Links work (including WHOP checkout)
- [ ] Mobile responsiveness
- [ ] Lighthouse score (aim for 90+ on all metrics)

### 8.3 Deploy to Vercel (Recommended)

**Option A: Vercel CLI**
```bash
npm i -g vercel
vercel
```

**Option B: GitHub Integration**
1. Push to GitHub
2. Import project in Vercel dashboard
3. Auto-detects Astro, configures build

### 8.4 Configure Custom Domain

1. Add domain in Vercel dashboard
2. Update DNS records:
   - A record → Vercel IP
   - Or CNAME → cname.vercel-dns.com
3. Enable HTTPS (automatic)

### 8.5 Alternative: Cloudflare Pages

```bash
npm run build
# Deploy dist/ folder to Cloudflare Pages
```
